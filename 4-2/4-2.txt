⚫️課題4-2
①ブロック要素とインライン要素の違いは何ですか？

→ブロック要素は、行全体のまとまり
 インライン要素は、行の中の一部のまとまり 


②ブロックレベル要素を10個以上上げなさい
 「div / p / h1~h6 / ul / hr / table / form / nav / header / footer / section / dl 」


③インライン要素を10個以上上げなさい
 「a / span / strong / u /mark / i / img / br / q / select / del / em」


④ブロックレベル要素、インライン要素の配置ルールは何ですか？
 
→ブロックレベル要素の中には、他のブロックレベル要素やインライン要素を配置することができる。
インライン要素の中には、文字データや他のインライン要素を配置できるが、インライン要素の中にブロックレベル要素を配置することはできない。


⑤高さ(height)と幅(width)指定できるのはブロック要素、インライン要素のどちらですか？
 
→ブロック要素


⑥marginに関してインライン要素での特徴を答えなさい。
 
→左右のみの margin 適用が可能で、上下への余白に対して適用を行うことはできない。


⑦paddingに関してインライン要素での特徴を答えなさい。
 
→余白を上下左右で適用することができるが、設定を行えても上下部分については、上下の要素の領域にはみ出してしまう。


⑧インライン要素とブロック要素の違いとしてプロパティの設定値の違いがあります。それは何のプロパティでしょうか。
 「 display プロパティ」
